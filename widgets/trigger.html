Trigger Binding

The trigger binding causes an event on the element, when the observable is set to a true value. The observable is then reset to false.

Syntax

trigger: {eventName: observable}

Code Example

<a data-bind="trigger: {click: causeClickEvent}" href="#linkwaspressed">The Target</a>
<button data-bind="click: causeClickEvent.set(true)">Pressing Me is Like Clicking That</button>
